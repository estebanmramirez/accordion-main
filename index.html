FAQs

What is Frontend Mentor, and how will it help me?

Frontend mentor offers realistic coding challenges to help
developers improve their frontend coding skills with projects in
HTML, CSS and Javascript. It's suitable for all levels and ideal for
portfolio building.

Is Frontend Mentor free?

In Frontend mentor you can see a lot projects to do, completely free
with the all resources that you need for complete the challenge.
Some others is paying that their difficulty is higher but you will
have more help to complete it.

Can I use Frontend Mentor projects in my portfolio?

Yes, you can. All the projects in Frontend mentor are a good
exercises to put in practice yours skills in HTML, CSS and
Javascript. You can upload your solution in Frontend mentor
or your Github profile.

How can I get help if I'm stuck on a challenge?

If you have the premium account you can chat with our
customer support to get the help that you need and
resolve the problem or have advice in your project.
